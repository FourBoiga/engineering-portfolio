# 16-bit CPU Design (Logisim)

## Overview

This project involves the design and implementation of a 16-bit CPU using Logisim. The system was built from fundamental digital logic components, including registers, an ALU, memory, and a control unit.

---

## Design Goal

Design a functional 16-bit processor capable of executing basic instructions, demonstrating an understanding of computer architecture and digital logic design.

---

## System Architecture

The CPU includes:

- Arithmetic Logic Unit (ALU)
- Register file
- Program counter (PC)
- Instruction decoding logic
- Memory interface
- Control unit

---

## Key Features

- 16-bit data width
- Custom instruction set
- ALU supporting arithmetic and logical operations
- Register-based architecture
- Program execution through instruction sequencing

---

## Engineering Understanding

The design was built from the ground up using logic gates and sequential components. Key concepts include:

- Combinational vs sequential logic
- Clocked state changes
- Instruction decoding and control signals
- Data flow between CPU components

Special attention was given to debugging signal flow and ensuring correct timing and state transitions during execution.

---

## Challenges

- Debugging incorrect data propagation between components
- Managing control signals across multiple subsystems
- Ensuring correct synchronization with the clock

---

## Current Status

- Core CPU design: complete
- Instruction execution: functional
- Further improvements possible in instruction set and efficiency
